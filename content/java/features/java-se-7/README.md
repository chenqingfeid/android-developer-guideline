## PermGen Removal

从 Java 7 开始，有一部分驻留在永生代的数据被移到了 Java 堆或 Native 堆中：

- 符号表移到了本地堆中
- Interned String 移到了 Java 堆中
- 类的静态成员移到了 Java 堆中
